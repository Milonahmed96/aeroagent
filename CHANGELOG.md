## [1.0.0] — 2025-03-06

### Added
- SQLite inventory database with 6 aviation parts and alternates
- LLM email extraction using Claude Sonnet (structured JSON output)
- Intelligent alternate part routing when primary is out of stock
- AI-generated professional quote emails
- Gradio web UI with sample emails
- Markdown code fence stripping for robust JSON parsing
- Full traceback error reporting in pipeline
